# SQL Revenue Reporting & Automation Project

This project demonstrates a SQL-based reporting solution that analyzes rental revenue by film category.
It transforms transactional payment data into summarized business metrics.

## Solution Overview
- Designed detailed and summary reporting tables
- Built a user-defined SQL function for revenue aggregation
- Implemented a trigger to keep summary data updated
- Created a stored procedure to refresh report data end-to-end

## Technologies Used
PostgreSQL, SQL (joins, aggregation, triggers, stored procedures)
